## Octave / Matlab Tutorial:
### Basic Operations
1. Elementary math operqations.

    ```
    >> % elementary math operations
    >> 3 + 5
    ans =  8
    >> 2 - 8
    ans = -6
    >> 2^3
    ans =  8
    >> 4/2
    ans =  2
    >>
    ```
2. Logical operations
    ```
    >> %some logical operation
    >> 1 == 2 % comparing two values
    ans = 0
    >> 1 ~= 2 % comparing two values with not equal to
    ans = 1
    >> 1 && 0 % AND
    ans = 0
    >> 1 || 0 % OR
    ans = 1
    >> xor(1,0)
    ans = 1
    >>
    ```
